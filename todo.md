- 결제할 금액을 먼저 장고 어드민에 등록되어 있는 것을 알려주고, 결제 시작 버튼 클릭하면 해당 사토시만큼의 라이트닝 인보이스 큐알 이미지와 인보이스 텍스트 보여줌. 모바일 화면의 경우에는 라이트닝 지갑 열기 버튼을 인보이스 텍스트 아래에 두어서 스마트폰에 있는 지갑 바로 열수 있도록 구성
결제 후 블링크 지갑에 api를 통해 성공 메시지를 받으면 '결제가 완료되었습니다.' 메시지 보여주고, 계약서 주소 생성 조건 충족
- 체크 박스 옆에 "계약 내용을 모두 확인했고, 자필 서명과 결제를 완료했습니다." 문구로 수정
- 라이트닝 결제 박스 영역에 시스템이 현재 어떤 진행 상황으로 라이트닝 결제를 처리하는지 텍스트로 계속 보여주도록 구성
- ln_payment 앱 기능 참고하고, 중간에 블링크 api 결제 처리 중 문제가 되면 왜 결제 처리가 되지 않았는지 안내 메시지를 보여줘서 유저가 혼란이 없도록 한다. 
- 생성된 인보이스는 60초가 유효하고, 인보이스 생성되면 카운트다운으로 60초에서 1초씩 줄어들도록 구성
- 60초 넘으면 인보이스 나오는 영역만 htmx로 새로고침해서 다시 결제 시작하도록 버튼 보이고,
- 인보이스 취소 버튼도 보여서 유저 스스로 취소하도록 한다. 취소하면 다시 결제 시작하도록 결제 시작 버튼 화면으로 이동
- 인보이스 확인 텀은 1초로 짧게 가지고 가자.
