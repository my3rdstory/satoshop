## 배경
- 타이틀/갤러리 텍스트 입력 길이가 짧아 확장이 필요했음.
- 매장 소개 줄바꿈이 랜딩/관리 화면에 반영되지 않았음.

## 변경 사항
- 타이틀 제목 1000자, 설명 10000자로 제한을 확장하고 관리 화면 안내를 갱신.
- 갤러리 설명 10000자로 제한을 확장하고 텍스트 영역 높이를 이미지와 맞추며 스크롤을 적용.
- 매장 소개 출력에 줄바꿈 렌더링을 적용.

## 영향
- 타이틀/갤러리 설명을 길게 입력할 수 있음.
- 갤러리 설명 입력 영역이 이미지 높이에 맞춰지고 긴 내용은 내부 스크롤로 표시됨.
- 매장 소개 줄바꿈이 랜딩/관리 카드에 반영됨.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 타이틀 제목 1000자/설명 10000자 입력 후 저장/미리보기.
2. 갤러리 아이템 설명에 10000자를 입력하고 텍스트 영역에 스크롤이 생기는지 확인.
3. 매장 소개에 줄바꿈을 포함해 입력 후 공개하고 `/minihome/<slug>/` 및 관리 카드에서 줄바꿈 표시를 확인.
