## 배경
- 공개 후 화면에 남아 있을 때 하단 알림으로 완료 여부를 확인할 수 있어야 했음.
- 새로고침 시 완료 알림이 반복 노출되는 문제를 막아야 했음.

## 변경 사항
- 공개 완료 시 좌측 하단 고정 알림 배너를 5초간 표시하도록 처리.
- 알림 표시 후 URL 쿼리에서 published 값을 제거해 새로고침 재노출을 방지.

## 영향
- 공개 완료 안내가 화면 방해 없이 확인되며, 재진입 시 반복 표시되지 않음.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 공개하기 클릭.
2. 좌측 하단에 공개 완료 알림이 표시되고 5초 후 사라지는지 확인.
3. 새로고침 시 알림이 다시 표시되지 않는지 확인.
