## 배경
- 매장 카드 높이를 텍스트 길이에 맞게 가변 처리하고, 지도 열기 버튼의 위치를 카드 하단으로 정렬해야 했음.
- CTA 섹션 이메일/후원 주소를 클릭하면 클립보드로 복사되도록 개선 요청이 있었음.

## 변경 사항
- 매장 카드 고정 높이를 제거하고, 텍스트 영역에 따라 카드 높이가 늘어나도록 수정.
- 지도 열기 버튼을 카드 하단 정렬되도록 레이아웃을 보완.
- CTA 이메일/후원 주소에 클릭 시 클립보드 복사 기능을 추가.

## 영향
- 매장 카드 높이가 소개 길이에 따라 유동적으로 변경됨.
- CTA 텍스트 클릭 시 복사 동작이 수행됨.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 매장 소개 길이를 다르게 입력 후 공개.
2. `/minihome/<slug>/`에서 매장 카드 높이가 텍스트에 따라 변하는지 확인.
3. 지도 열기 버튼이 카드 하단에 위치하는지 확인.
4. CTA 이메일/후원 주소 클릭 시 클립보드에 복사되는지 확인.
