## 배경
- 미니홈 관리 화면에서 데이터 보기를 눌러도 새로 고침 시 선택한 데이터 영역이 닫혀 불편했음.

## 변경 사항
- 데이터 보기 선택 시 섹션 ID를 URL 파라미터로 반영.
- 페이지 로드 시 URL 파라미터를 읽어 마지막으로 열었던 데이터 패널을 복원.

## 영향
- 새로 고침 후에도 마지막으로 열었던 데이터 패널이 유지됨.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 데이터 보기 버튼 클릭.
2. 주소창에 `data_panel` 파라미터가 추가되는지 확인.
3. 새로 고침 후 동일 데이터 패널이 열려 있는지 확인.
