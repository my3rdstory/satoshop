## 배경
- 미니 블로그 글을 제목/본문으로 분리하고 노출 규칙을 명확히 할 필요가 있었음.
- 컨트리뷰터 섹션을 추가해 썸네일·닉네임·링크를 노출해야 했음.

## 변경 사항
- 미니 블로그를 제목/본문 구조로 변경하고 제목 100자 제한과 카드/모달 표시 로직을 반영.
- 컨트리뷰터 섹션을 추가해 썸네일 100px 정사각 WebP 변환, 닉네임/링크 입력, 랜딩 카드 노출을 지원.
- 관리 화면에 컨트리뷰터 추가/편집 UI와 필수 입력 검증을 추가.
- 미니 블로그 구형 데이터 렌더링 시 `text` 참조로 인한 오류가 나지 않도록 템플릿 표시를 정리.
- 컨트리뷰터 카드의 썸네일을 원형으로 변경하고 닉네임을 가운데 정렬로 맞춤.
- 컨트리뷰터 카드 호버 효과를 제거하고 썸네일에 줌 효과를 추가.
- 컨트리뷰터 카드 박스 스타일을 제거해 썸네일/닉네임만 노출되도록 조정.
- 컨트리뷰터 카드 묶음을 좌측 정렬로 맞춤.
- 컨트리뷰터 세트가 좌측에서 오른쪽으로 쌓이도록 카드 폭을 고정.
- 미니 블로그 제목이 카드 영역을 넘지 않도록 말줄임 처리를 보강.
- 미니 블로그 썸네일 비율을 모달 이미지 비율에 맞게 조정.

## 영향
- 미니홈 카드에는 제목만 노출되고, 클릭 시 모달에서 제목/본문/이미지를 확인할 수 있음.
- 컨트리뷰터 카드가 화면 너비에 맞춰 자동 줄바꿈으로 배치됨.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 미니 블로그 제목/본문/이미지를 입력한 뒤 저장 및 미리보기로 카드에 제목만 보이는지, 클릭 시 모달에서 제목/본문/이미지가 보이는지 확인.
2. 컨트리뷰터 섹션을 추가하고 썸네일/닉네임/링크를 입력한 뒤 저장/미리보기로 카드가 자동 줄바꿈되는지와 이미지/닉네임 클릭 시 새 탭 링크가 열리는지 확인.
3. 컨트리뷰터 썸네일 업로드 시 100px 정사각 WebP 변환이 반영되는지 확인.
