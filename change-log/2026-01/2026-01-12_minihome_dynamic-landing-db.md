## 배경
- Render 배포 시 정적 미니홈 파일이 초기화되어 공개 페이지가 리셋되는 문제가 있었음.

## 변경 사항
- 미니홈 랜딩을 정적 파일 대신 DB의 공개 섹션으로 렌더링하도록 전환.
- 도메인 매핑도 DB 기준으로 조회해 공개된 미니홈만 노출.
- 공개하기 버튼은 DB 업데이트만 수행하고 정적 파일 생성은 생략.

## 영향
- 배포/재시작 시에도 미니홈 공개 데이터가 유지됨.
- 공개 미니홈은 항상 DB 최신 데이터로 렌더링됨.

## 대장 수동 테스트 가이드
1. `/minihome/<slug>/mng/`에서 공개하기 후 `/minihome/<slug>/`에 최신 내용이 보이는지 확인.
2. Render 재배포 후에도 동일 미니홈 내용이 유지되는지 확인.
3. 도메인 매핑이 설정된 경우 도메인 루트에서 공개 미니홈이 노출되는지 확인.
