# api · 공지사항 조회 API 추가

## 요약
- 공지사항 목록/상세를 조회하는 외부 API를 추가했습니다: `GET /api/v1/notices/`, `GET /api/v1/notices/{notice_id}/`.
- 목록은 활성 공지사항을 고정 여부·작성일 역순으로 반환하며 제목/본문/고정 여부/생성·수정 시각을 포함합니다.
- 상세는 제목/본문/작성자/고정 여부 및 생성·수정 시각을 제공해 외부 앱에서 공지 본문을 바로 노출할 수 있습니다.
- Swagger 정적 스펙(`api/static/api/openapi-v1.json`)과 README 외부 API 섹션을 함께 갱신했습니다.
- Swagger UI에서 공지 GET 항목을 스토어 목록보다 상단에 노출하도록 경로 순서를 조정했습니다.
