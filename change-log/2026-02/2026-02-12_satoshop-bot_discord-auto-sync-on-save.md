## 배경
- 디스코드 슬래시 명령어를 수동으로 동기화하지 않으면 테스트 채널에서 즉시 명령어가 보이지 않는 문제가 반복되었습니다.
- 봇 설정 저장 시 자동 동기화가 필요했습니다.

## 변경 사항
- `sync_discord_commands --all-guilds`를 실행해 글로벌/길드 명령어를 즉시 동기화했습니다.
- `DiscordBot` 저장(`post_save`) 시 자동으로 명령어 동기화를 수행하는 시그널을 추가했습니다.
  - 조건: `is_active=True` 및 `application_id`, `token`이 설정된 경우
  - 동작: 봇이 가입한 길드 목록 조회 후 `글로벌 + 길드` 동시 동기화
- 자동 동기화 동작을 README에 문서화했습니다.

## 영향
- 어드민에서 봇 설정 저장 후 별도 명령 실행 없이 슬래시 명령어 동기화를 자동 시도합니다.
- 운영/테스트 길드에서 명령어 반영 누락 가능성이 줄어듭니다.

## 대장 수동 테스트 가이드
1. 어드민에서 `application_id`, `public_key`, `token`을 확인한 뒤 저장.
2. 개발 서버 로그에서 자동 동기화 성공 로그(`global`, `guilds`) 확인.
3. 디스코드 길드 아무 채널에서 `/사토샵_최근등록상품`, `/사토샵_최근판매상품` 자동완성 노출 확인.
4. 명령 실행 결과가 실행자 본인에게만 보이는지(에페메랄) 확인.
