# Expert 계약 PDF 폰트 번들 추가

## 배경
- 서버 환경마다 한글 폰트가 설치되지 않아 계약/채팅 PDF가 여전히 깨진다는 제보가 있어 레포 차원에서 일관된 폰트를 제공할 필요가 생김.

## 변경 사항
- SIL OFL 라이선스의 `NanumGothic-Regular.ttf`를 `expert/fonts/`에 번들로 추가하고, ReportLab 폰트 우선순위를 이 파일 → 시스템 폰트 → CID 폰트 순으로 재구성.
- README의 한글 폰트 안내를 업데이트하여 번들 폰트 사용 방식과 커스터마이징 방법을 명시.

## 검증
- 계약/채팅 PDF 생성 시 본문과 해시/서명 텍스트에 한글이 포함된 예시를 만들어 PDF를 열어 한글이 올바르게 표시되는지 확인한다.
