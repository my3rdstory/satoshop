# Expert 계약 미리보기 기능 제거

## 배경
- 계약 초안 화면의 미리보기 카드/모달이 최종 계약서 PDF와 일치하지 않아 혼란을 준다는 피드백을 반영해 기능을 제거하고 단순 입력 흐름으로 정리했습니다.

## 변경 사항
- `contract_draft.html`에서 우측 미리보기 카드와 모달, 관련 스크립트 자원(마크다운 렌더러) 호출을 제거하고 단일 컬럼 입력 UI로 재구성.
- 동일 화면 상단에 현재 노출 중인 표준 계약서 정보를 간단한 알림으로 안내하고, 템플릿 미등록 시 경고 메시지를 표시.
- 랜딩 페이지(`direct_contract_start.html`)와 README에서 더 이상 미리보기 기능을 언급하지 않도록 설명을 수정.

## 검증
- `/expert/contracts/direct/draft/` 접속 시 미리보기 카드/모달이 노출되지 않고 입력 폼만 표시되는지 확인.
- 템플릿이 존재할 때 알림 문구가 올바르게 표시되고, 템플릿이 없으면 경고가 노출되는지 확인.
