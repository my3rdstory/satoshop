## 개요
- Expert 최종 계약서 PDF에서 표 셀 내부 줄바꿈이 하나의 줄로 뭉개지던 문제를 해결하고, 동작 방식을 README에 명확히 기록했습니다.

## 상세
1. Markdown 표 파싱 시 각 셀 내용을 fpdf2 테이블 렌더러에 맞춰 `<br>`/줄바꿈을 HTML로 치환하고, 셀 높이를 자동 계산해 다단 줄바꿈이 그대로 렌더링되도록 처리했습니다.
2. PDF 문서화 섹션에 “표 셀 줄바꿈 보존” 항목을 추가해 계약 조건 입력자가 기대하는 출력 형태를 안내합니다.
3. HTML 모듈 이름과 동일한 지역 변수가 충돌해 PDF 미리보기에서 NameError가 발생하지 않도록 모듈 레퍼런스를 별도 이름으로 분리하고, 표 셀 파싱을 반복문 기반으로 바꿔 클로저 의존성을 제거했습니다.
4. 테이블 셀 문자열뿐 아니라 작업 로그 등 모든 다중 줄 필드에서 `\r\n`, `\n`, JSON 리터럴 `\\n`을 실제 개행으로 복원한 뒤 fpdf2 `multi_cell`로 렌더링해 줄바꿈/연속 공백이 그대로 유지되도록 했습니다.
5. 수행 내역 섹션은 하나의 줄바꿈만 Markdown `<br/>`로 처리하고, 둘 이상 연속될 때는 `&nbsp;` 단독 문단을 삽입한 Markdown 블록으로 분리하며, 3줄 이상 연속 줄바꿈은 자동으로 2줄 여백으로 축약합니다.
