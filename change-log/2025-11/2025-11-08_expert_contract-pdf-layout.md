# Expert 계약 PDF 레이아웃 개선

## 배경
- 최종 계약서 PDF 상단에 `json.dumps` 결과와 Markdown 원본이 그대로 노출되어 문서 가독성이 떨어지고 일반 계약서 양식처럼 보이지 않는 문제가 있었음.

## 변경 사항
- 계약 개요 섹션을 사람이 읽기 좋은 문장으로 재구성하고, 역할/기간/금액/지급 방식/라이트닝 주소/작업 메모 등을 항목별로 출력.
- Markdown 본문을 간단한 텍스트 라인(제목, 목록, 인용, 코드)으로 변환하는 파서를 추가해 `#, -, 1.` 같은 마크가 그대로 노출되지 않도록 개선.
- 분할/일괄 지급 내역도 별도 섹션으로 묶어 조건/일자를 보기 쉽게 배치하고, 모든 섹션 출력 로직을 `_write_lines` 헬퍼로 정리해 페이지 넘김 처리를 일관화.

## 검증
- Markdown 헤더/목록/인용이 포함된 계약서 템플릿으로 계약 PDF를 생성하면 각 요소가 텍스트 형태로 깔끔하게 표현되는지 확인.
- 분할 지급, 일괄 지급 케이스 각각 PDF를 생성해 섹션이 올바른지 확인.
