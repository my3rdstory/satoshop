# 어드민 임시 비밀번호로 비밀번호 변경 허용

## 변경 내용
- 어드민의 `임시 비밀번호`로 로그인한 사용자가 마이페이지 비밀번호 변경 화면에서 같은 값을 `현재 비밀번호`로 입력할 수 있도록 수정했다.
- 임시 비밀번호를 사용해 새 비밀번호로 변경이 완료되면 저장된 임시 비밀번호는 자동 제거되도록 정리했다.
- 어드민 도움말과 README에 임시 비밀번호 동작 방식을 문서화했다.

## 영향 범위
- 계정 비밀번호 변경 폼 검증
- 어드민 사용자 상세의 임시 비밀번호 안내
- 계정 기능 문서

## 수동 검증 메모
- 장고 어드민 사용자 상세에서 `임시 비밀번호`를 설정한다.
- 해당 사용자로 로그인한다.
- `http://현재호스트:8011/accounts/change-password/`에서 현재 비밀번호에 임시 비밀번호를 입력하고 새 비밀번호로 변경한다.
- 변경 후 로그아웃 뒤 새 비밀번호로 재로그인되는지 확인한다.
