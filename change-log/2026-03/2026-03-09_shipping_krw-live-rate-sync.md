# 스토어 배송비 원화 환율 연동 통일

## 변경 내용
- 스토어 배송 설정의 `배송비(원화)`와 `무료 배송 기준(원화)`를 저장 시점 고정 사토시값이 아니라 조회/주문 계산 시점의 최신 BTC/KRW 환율로 다시 계산하도록 수정했다.
- 상품 등록 화면의 원화 연동과 동일한 환율 소스(`UpbitExchangeService` / 최신 `ExchangeRate`)를 배송비 계산에도 사용하도록 통일했다.
- 배송비 설정 화면과 README에 원화 기준 배송비가 최신 환율로 계속 재계산된다는 안내를 추가했다.

## 영향 범위
- 스토어 배송비 설정 화면
- 상품 상세/목록의 배송비 표기
- 장바구니/주문 생성 시 배송비 계산

## 수동 검증 메모
- `/stores/<store_id>/edit/shipping-settings/`에서 배송비와 무료 배송 기준을 원화로 저장한다.
- `/products/<store_id>/add/`에서 배송 정책 안내의 원화/사토시 표기가 노출되는지 확인한다.
- 환율 데이터가 갱신된 뒤 상품 상세/장바구니/체크아웃에서 배송비 사토시 금액이 최신 환율 기준으로 바뀌는지 확인한다.
