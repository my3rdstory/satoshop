# README 최신 코드 기준 재구성

## 요약
- 현재 코드베이스를 기준으로 README를 전면 재정리했다.
- 전자상거래, 인증, 결제, 미니홈, Discord 봇, Expert 계약, 외부 API 등 실제 운영 중인 시스템 구성을 문서에 반영했다.

  - 설치 후 관리자, 판매자, 외부 연동 담당자가 바로 활용할 수 있는 운영 흐름 정리
