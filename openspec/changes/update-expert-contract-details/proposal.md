## 배경
- Expert 직접 계약 플로우에서 계약 후 이메일 발송에 사용할 Gmail 자격 증명을 어드민에서 입력/관리할 방법이 필요합니다.
- 계약 드래프트/미리보기 UX 중 "중개자" 표기, 분할 지급 세부 입력, 수행 내역 메모 노출이 부족해 사용자가 세부 항목을 확인하기 어렵습니다.
- 계약 첨부는 PDF만 허용해야 하지만 현재 클라이언트 신뢰에 의존하고 있으며, 미리보기·최종 문서에서 첨부 내용이 누락되어 있습니다.

## 목표
1. Expert 이메일 설정을 장고 어드민에서 최초 등록/수정할 수 있도록 하고, Gmail 기반 발송 정보를 노출한다.
2. 계약 드래프트 화면의 용어를 "중개자"로 정정하고, 분할 지급 선택 시 각 회차의 금액·지급일·조건을 입력/검증하여 요약/미리보기에 반영한다.
3. 수행 내역 메모를 계약서 미리보기 "2. 계약 내용" 블록에 포함하고, 첨부된 PDF 목록을 미리보기·검토·초대 화면 및 최종 PDF 하단에 붙임 형태로 출력한다.
4. 첨부 파일이 서버 측에서도 PDF인지 검증하고, 최종 계약서 미리보기/문서에서 업로드된 파일을 내려받을 수 있게 링크를 제공한다.

## 영향 범위
- `expert` 앱의 어드민, 폼/뷰/템플릿, 정적 JS/CSS, PDF 생성 로직.
- `myshop.SiteSettings` Proxy(admin)에서 Gmail 설정 추가/저장 방식.

## 제외 사항
- 실제 Gmail 발송 테스트나 외부 SMTP 연동 방식 변경.
- 첨부 파일 다중 포맷 지원 또는 미리보기 Embed.
- 계약 저장/이메일 발송 파이프라인 구조 개선.
